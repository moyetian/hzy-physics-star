# GSD Work Log

## 2026-06-29

### Goal

把学生纪念册从展示型网页升级为可反复互动的学习纪念册，同时保持零依赖、静态部署、PWA 离线可用。

### Scope

- 保留原有星空、线条小狗、物理星球、毕业信和未来信箱。
- 新增实用互动功能，所有数据只保存在浏览器本地。
- 增强学习相关小游戏，覆盖高中物理主要知识点。
- 将 10 个高中常考实验做成可视化模型，而不是单一滑杆演示。

### Execution

- 新增 `星光工具箱`：
  - 每日幸运公式和小任务。
  - 可复制的今日星光卡。
  - 星光纪念清单。
  - 给未来自己的本地便签，使用 `localStorage` 自动保存。
- 新增 `物理游乐场`：
  - `物理快问星`：200 道高中物理快问题，每轮随机抽 10 道。
  - `公式配对机`：200 张概念/公式配对卡。
  - `趣味实验室`：10 个高中常考实验。
- 实验可视化模型：
  - 牛顿第二定律小车。
  - 胡克定律弹簧。
  - 欧姆定律电路。
  - 电源电动势和内阻。
  - 单摆测 g。
  - 凸透镜成像。
  - 抛体运动。
  - 动量守恒碰碰车。
  - 安培力磁场实验。
  - 光电效应真空管模型。
- 交互联动：
  - 小狗、星球、读信、未来信箱、便签、学习小游戏都会写入星光纪念清单。
  - 实验切换和滑杆参数变化会实时重绘对应模型。
- PWA 缓存版本更新到 `v13`，避免浏览器继续使用旧资源。
- README 同步记录当前功能和后续方向。

### Risks

- 当前没有使用真实浏览器自动化测试工具，视觉和交互细节仍需人工浏览器验证。
- 题库覆盖面已经扩展，但题目仍应由老师后续抽样审校，避免表述不严谨。
- 可视化模型是教学示意，不是精密仿真；应避免把动画比例当作严格实验数据。
- PWA 可能受旧 service worker 缓存影响，验证新版时需要 `Ctrl + F5` 或连续刷新。
