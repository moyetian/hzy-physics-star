# 给何芷滢同学的物理星球毕业纪念册

一个零依赖静态网页原型，风格为可爱治愈、星空梦幻，并使用原创线条小猫小狗作为主视觉。内容包含首页、生日倒计时、五颗物理祝福星球、毕业短信、未来信箱和 PWA 离线缓存。

## 本地预览

```bash
python3 -m http.server 8000
```

然后打开：

```text
http://127.0.0.1:8000/he-zhiying-physics-star/
```
