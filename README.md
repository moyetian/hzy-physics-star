# 给何芷滢同学的物理星球毕业纪念册

一个零依赖静态网页原型，风格为可爱治愈、星空梦幻，并使用原创线条小猫小狗作为主视觉。内容包含首页、生日倒计时、星光工具箱、物理学习小游戏、五颗物理祝福星球、毕业短信、未来信箱和 PWA 离线缓存。

## 已有功能

- 线条小狗互动、换悄悄话、爪印和星光特效。
- 生日天数统计和下一个生日倒计时。
- 可点击的物理星球轨道图。
- 星光工具箱：每日幸运公式、纪念清单、给未来自己的本地便签。
- 物理游乐场：200 道高中物理快问题、200 张公式配对卡、10 个高中常考趣味实验；其中小车、弹簧、单摆、抛体、碰撞、磁场力使用 Canvas 2D 刚体/约束模型，并按固定时间步、世界坐标比例尺和课堂公式驱动动画；欧姆定律、折射与全反射、光电效应使用专门的电路/光路/能量阈值模型。
- 复习陪伴：错题星云、知识点地图和低调学习成就，全部保存在当前设备。
- 情绪小屋：情绪天气、倾诉空间、呼吸星球、小狗陪伴、星星泡泡、整理云朵、情绪记录和求助小纸条，全部只保存在当前设备，不做情绪诊断。
- 小数据抽屉：导出本地记录，或清空学习记录、情绪小屋记录和全部本地数据。
- 未来信箱，按日期逐步解锁。
- PWA 离线缓存，适合部署后添加到桌面。

## 脚本结构

- `data.js`：星球、星光卡、题库、公式卡、实验定义和文案数据。
- `learning.js`：错题星云、知识点地图和学习成就。
- `mood.js`：情绪小屋。
- `lab.js`：物理实验渲染和 Canvas 模型。
- `main.js`：DOM 引用、通用工具、页面初始化和事件绑定。
